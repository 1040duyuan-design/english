# 资源映射规则（v4）

## 核心公式

技能 × 问题类型 × 目标场景

## 设计原则

### 1. 不能只按技能分难度
同样都是口语弱，用户可能是：

- 词汇少
- 发音不自然
- 反应慢
- 会想不会说

所以资源不能只写“口语简单版”。

### 2. 资源不只分难度，还要分训练方式
例如：

- 发音自然度偏弱 → 跟读 + 重音节奏
- 开口反应慢 → 限时开口 + 场景抢答
- 听力输入薄弱 → 慢速输入 + 关键词抓取
- 阅读强于输出 → 半开放写作 + 角色替换表达

### 3. 先给第一周路径，再做二次校准
校准信号包括：

- 完成率
- 跳过率
- 自我反馈
- 阶段复测结果

## 难度带

### 基础支撑版
适合分数较低或基础混乱用户。
重点是高频词、句型骨架、慢速输入、强引导。

### 过渡训练版
适合已有一定基础但应用不稳定的用户。
重点是标准场景材料、半开放输出、正常语速输入。

### 挑战提升版
适合基础较稳、需要接近真实应用的用户。
重点是更真实的材料、更少支架、更强调快速反应。
